# Google Maps Reviews Scraper Output Fields

## Output Overview

`Google Maps Reviews Scraper` returns one row per review.

This is a `review-level` template, not a business-level template.

## High-Value Output Fields

| Field | Meaning |
|---|---|
| `Input` | The input place URL or source identifier |
| `name` | Place name |
| `Rating` | Place rating summary |
| `num_review` | Total review count summary |
| `author_title` | Reviewer name |
| `author_link` | Reviewer profile link when available |
| `is_local_guide` | Whether the reviewer is a Local Guide |
| `num_author_reviews` | Total review count by that reviewer |
| `author_rating` | Rating given by the reviewer |
| `author_review_timestamp` | Review date or timestamp |
| `review_text` | Main review text |
| `review_img_url` | Review image URL when available |
| `review_likes` | Like count on the review |
| `review_link` | Direct review link when available |

## Practical Output Uses

Use these outputs for:

- review sentiment analysis
- reviewer-quality research
- top-location comparisons
- Google Maps reputation monitoring
- competitor review benchmarking

## Important Output Note

This template is not meant to replace a business discovery template.

It enriches business records with review-level evidence.
